
# WeatherApp

This app is in development now



## Features

- Fullscreen mode
- Widget mode
- Current weather by geo
- Eng/rus language
- search cities(needs improvement)
- saved cities list(needs improvement)


## Tech Stack

- Kotlin
- MVVM
- Glide
- Room
- Widget
- Jetpack compose
- Jetpack navigation
